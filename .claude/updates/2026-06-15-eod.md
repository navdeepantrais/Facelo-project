## 1. Category Carousel — Centering Fix

**File:** `app/CategoryCarousel.tsx`

**Problem:** On wide desktop viewports (1536px+) where all 9 category cards fit within the container, cards were left-aligned with empty space on the right. Two prior attempts failed: `md:justify-center` breaks scroll UX (symmetrical clipping), `auto-fill` grid left empty column tracks causing left alignment.

**Fix:** `w-max mx-auto` pattern on the inner flex row inside an `overflow-x-auto` outer wrapper.
- Inner div at exactly its intrinsic width (`w-max`), centered via `mx-auto`
- When content fits: auto margins center the row ✓
- When content overflows: auto margins resolve to 0, scroll starts from the left edge ✓
- Works at all breakpoints — no mobile/tablet/desktop split needed

**Also:** Removed `'use client'` directive and `useHorizontalScroll` hook — component is now a pure Server Component. No hooks, no event handlers, no browser APIs needed.
